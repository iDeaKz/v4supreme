## **🔧 TROUBLESHOOTING**

### **❌ Server Won't Start**
```bash
# Check if port 8000 is in use:
netstat -ano | findstr :8000

# Kill existing processes:
taskkill /f /im python.exe
```

### **❌ Pages Won't Load**
```bash
# Check server status:
curl http://localhost:8000

# Restart server:
python -m http.server 8000
```

### **❌ Browser Issues**
- Clear browser cache
- Disable ad blockers
- Enable JavaScript
- Try incognito mode
